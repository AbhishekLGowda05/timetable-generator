### A1. Generated Timetable Is Not Retrievable

Even after job creation:

```

```

Remaining gap:

* Frontend never calls `GET /result/{job_id}` after job completes.
* No polling loop bound to result hydration.

```
        └── if DONE → fetch result → store → navigate /results/{job_id}
```

Before solver:

```
   teacher_capacity

   resource_capacity
```

### B2. No Proof Soft Constraints Are Actually Soft

Likely mistake pattern:

```
model.Add(condition)     # HARD
instead of
model.Add(condition).OnlyEnforceIf(penalty.Not())
```

If so → all “soft” constraints are hard.

Weights currently raw numbers.

Without normalization:

```

```

Should use:

```

```

and comparable scale across constraints.
